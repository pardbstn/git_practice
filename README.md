## 개인과제 필수문제 1

<details>
  <summary> 과제 상세보기 </summary>

**제시된 예시 점수를 기준으로 조건에 맞는 등급을 판별하여 출력하는 프로그램 작성하기.**

### ✔ 과제 설명
예시 점수: 84점

등급 기준

90점 ~ 100점 → A등급

80점 ~ 89점 → B등급

80점 미만 → C등급

출력 형태:
이 학생의 점수는 [점수] 이며, 등급은 [등급]입니다!

### ✔ 구현 내용
변수 선언

int score = 84; → 학생 점수 저장

String grade; → 계산 후 등급 저장

조건문(if~else)

점수가 90~100이면 grade = 'A등급'

점수가 80~89이면 grade = 'B등급'

그 외에는 grade = 'C등급'

결과 출력

문자열 보간을 사용해
print('이 학생의 점수는 $score 이며, 등급은 $grade입니다!');
형태로 결과 출력

### ✔ 핵심 코드 예시

```dart
void main() {
  int score = 84;
  String grade;

  if (score >= 90 && score <= 100) {
    grade = 'A등급';
  } else if (score >= 80 && score < 90) {
    grade = 'B등급';
  } else {
    grade = 'C등급';
  }

  print('이 학생의 점수는 $score 이며, 등급은 $grade입니다!');
}
```

## 개인과제 필수문제 2

<details>
  <summary> 과제 상세보기 </summary>

  **쇼핑몰 상품 가격을 기반으로 장바구니의 총 금액을 계산하고,  
2만원 이상일 경우 10% 할인 적용하는 프로그램입니다.**

### ✔ 과제 설명
- Map을 이용해 상품명과 가격을 저장하고  
- List로 장바구니를 구성한 뒤  
- for문을 사용해 총 결제 금액을 계산  
- 총합이 **20,000원 이상이면 10% 할인**  
- 할인 금액과 최종 결제 금액을 출력하는 프로그램

### ✔ 구현 내용
- `Map<String, int>` : 상품명 → 가격  
- `List<String>` : 장바구니 아이템  
- for-in 반복문으로 장바구니에 담긴 상품 가격 합산  
- 조건문(if/else)로 할인 적용 여부 판별  
- 2만원 이상 → 10% 할인 적용 (double 타입)

### ✔ 핵심 코드 예시

```dart
void main() {
  print("필수과제 2 - 쇼핑몰 만들기");

  // 쇼핑몰 품목 (Map)
  Map<String, int> shop = {
    '티셔츠': 10000,
    '바지': 5000,
    '모자': 3000,
    '가방': 50000
  };

  // 장바구니 (List)
  List<String> cart = ['티셔츠', '바지'];

  // 총합 계산
  int total = 0;
  for (String item in cart) {
    total += shop[item]!;
  }

  // 2만원 이상 → 10% 할인
  if (total >= 20000) {
    double discount = total * 0.9;
    double minus = total * 0.1;

    print("이 상품은 10% 할인된 가격으로 총 $discount원이 들어있습니다.");
    print("❤️ 고객님께 총 $minus원을 할인해드렸습니다 ❤️");
    print(" ");
    print("구매해주셔서 감사합니다 🥰");
  }
  // 2만원 미만 → 할인 없음
  else {
    print("이 장바구니에는 총 $total원이 들어있습니다.");
    print("구매해주셔서 감사합니다 🥰");
  }
}
```
