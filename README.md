## 개인과제 필수문제 1

<details>
  <summary> 과제 상세보기 </summary>

**제시된 예시 점수를 기준으로 조건에 맞는 등급을 판별하여 출력하는 프로그램 작성하기.**

### ✔ 과제 설명
예시 점수: 84점

등급 기준

90점 ~ 100점 → A등급

80점 ~ 89점 → B등급

80점 미만 → C등급

출력 형태:
이 학생의 점수는 [점수] 이며, 등급은 [등급]입니다!

### ✔ 구현 내용
변수 선언

int score = 84; → 학생 점수 저장

String grade; → 계산 후 등급 저장

조건문(if~else)

점수가 90~100이면 grade = 'A등급'

점수가 80~89이면 grade = 'B등급'

그 외에는 grade = 'C등급'

결과 출력

문자열 보간을 사용해
print('이 학생의 점수는 $score 이며, 등급은 $grade입니다!');
형태로 결과 출력

### ✔ 핵심 코드 예시

```dart
void main() {
  int score = 84;
  String grade;

  if (score >= 90 && score <= 100) {
    grade = 'A등급';
  } else if (score >= 80 && score < 90) {
    grade = 'B등급';
  } else {
    grade = 'C등급';
  }

  print('이 학생의 점수는 $score 이며, 등급은 $grade입니다!');
}
```

## 개인과제 필수문제 2

<details>
  <summary> 과제 상세보기 </summary>

  가상의 쇼핑몰에서 결제 금액을 계산하는 프로그램을 만들어봅니다.

1. 구매자가 장바구니에 담은 상품들의 전체 금액을 계산합니다.
2. 총 금액이 20,000원 이상일 경우, 10% 할인을 적용합니다.
3. 최종적으로 구매자가 결제해야 하는 금액을 출력합니다.

### ✔ 과제 설명
- Map을 이용해 상품명과 가격을 저장하고  
- List로 장바구니를 구성한 뒤  
- for문을 사용해 총 결제 금액을 계산  
- 총합이 **20,000원 이상이면 10% 할인**  
- 할인 금액과 최종 결제 금액을 출력하는 프로그램

### ✔ 구현 내용
- `Map<String, int>` : 상품명 → 가격  
- `List<String>` : 장바구니 아이템  
- for-in 반복문으로 장바구니에 담긴 상품 가격 합산  
- 조건문(if/else)로 할인 적용 여부 판별  
- 2만원 이상 → 10% 할인 적용 (double 타입)

### ✔ 핵심 코드 예시

```dart
void main() {
  // 상품 가격 정보
  Map<String, int> prices = {
    '티셔츠': 10000,
    '바지': 8000,
    '모자': 4000,
  };

  // 장바구니 목록
  List<String> cart = ['티셔츠', '바지', '모자', '티셔츠', '바지'];

  // 1번: 총 금액 계산
  int total = 0;
  for (String item in cart) {
    total += prices[item]!;
  }

  print('장바구니에 $total원 어치를 담으셨네요!');

  // 2번: 할인 계산 (10%)
  int discount = 0;
  if (total > 20000) {
    discount = (total * 0.1).toInt(); 
    print('할인 금액 : $discount원');
  }

  // 3번: 최종 결제 금액 계산
  int finalPrice = total - discount;

  print('최종 결제 금액은 ${finalPrice}원입니다!');
}

```
